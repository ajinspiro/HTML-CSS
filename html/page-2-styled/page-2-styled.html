We use CSS for telling the browser how HTML is rendered.
There are 3 ways to write CSS.
  • Inside <style></style> tag.
  • In a separate text file with .css extension. You have to link that file in <head> section using a <link/> tag.
    Link tag has 2 attibutes, 'rel' and 'href'. The 'rel' must be set to "stylesheet"
    and 'href' must be set to the location of your .css file.
    e.g : <link rel="stylesheet" href="page-2-styled.css" />
  • Using style="" attribute of html elements.
    The container div tag of this text has some styles applied to it in this way.
Syntax of CSS ruleset
selector {
property: value;
}

dot (e.g, .link-tag-example ) is used to select elements with class attribute.
hash (e.g, #calculate ) is used to select an HTML element with id attribute
